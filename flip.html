CSS 3D Properties – Easy Notes

1. perspective

Definition:
perspective defines how far the object is from the viewer. It gives a 3D depth effect.

Smaller value → stronger 3D effect
Bigger value → flatter effect

Syntax:
perspective: 800px;

Where we use it?

  • Card flip animations
  • 3D sliders
  • Image hover effects
  • Gaming UI

2. transform-style: preserve-3d

Definition:
preserve-3d tells the browser to keep child elements in 3D space.

Without it → child elements become flat (2D)
With it → real 3D rotation works

Syntax:
transform-style: preserve-3d;

Where we use it?

  • Flip cards
  • 3D boxes
  • Cube animations
  • Carousel effects

3. backface-visibility

Definition:
backface-visibility controls whether the backside of an element is visible when rotated.

Values:

  • visible → backside shows
  • hidden → backside hides

Syntax:
backface-visibility: hidden;

Where we use it?

  • Card flip animations
  • Login/Register flip forms
  • Quiz apps
  • Product detail cards
