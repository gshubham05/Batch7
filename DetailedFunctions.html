JavaScript Functions

What is a Function?

A function is a block of code that performs a specific task. It runs only when it is called.

Example: Making tea again and again → write steps once → call whenever needed.

Parts of a Function

1) Function Definition

Writing the function code is called function definition.

function greet(){ alert("Hello"); }

2) Function Call

Using the function is called function calling.

greet();

Important Terms

  • Parameter: Variable used while defining function
  • Argument: Value passed while calling function
  • Return: Sends value back from function

4 Types of Functions

1) Simple Function (No Parameter, No Return)

Only performs task.

function hello(){ console.log("Hello World"); } hello();

10 Examples

function ex1(){ console.log("One"); } ex1(); function ex2(){ alert("Two"); } ex2(); function ex3(){ console.log(5+5); } ex3(); function ex4(){ document.write("Hello"); } ex4(); function ex5(){ console.log("JS"); } ex5(); function ex6(){ console.log(true); } ex6(); function ex7(){ console.log(100); } ex7(); function ex8(){ console.log("Learning"); } ex8(); function ex9(){ console.log("Function"); } ex9(); function ex10(){ console.log("Done"); } ex10();

2) Function with Return (No Parameter)

Returns value but does not take input.

10 Examples

function a(){ return 10; } console.log(a()); function b(){ return "Hello"; } console.log(b()); function c(){ return 5+5; } console.log(c()); function d(){ return true; } console.log(d()); function e(){ return 100; } console.log(e()); function f(){ return "JS"; } console.log(f()); function g(){ return 50-20; } console.log(g()); function h(){ return 2*5; } console.log(h()); function i(){ return "Done"; } console.log(i()); function j(){ return 1; } console.log(j());

3) Function with Return & Parameter

Takes input and returns output.

10 Examples

function add(a,b){ return a+b; } console.log(add(2,3)); function sub(a,b){ return a-b; } console.log(sub(5,2)); function mul(a,b){ return a*b; } console.log(mul(2,4)); function div(a,b){ return a/b; } console.log(div(10,2)); function sq(a){ return a*a; } console.log(sq(5)); function cube(a){ return a*a*a; } console.log(cube(3)); function greet(n){ return "Hello "+n; } console.log(greet("Ram")); function age(a){ return a>=18; } console.log(age(20)); function len(s){ return s.length; } console.log(len("JS")); function double(x){ return x*2; } console.log(double(6));

4) Function without Return but With Parameter

Takes input but does not return anything.

10 Examples

function show(a){ console.log(a); } show(10); function msg(n){ alert("Hi "+n); } msg("Aman"); function sum(a,b){ console.log(a+b); } sum(5,5); function print(s){ document.write(s); } print("Hello"); function even(n){ console.log(n%2==0); } even(4); function sqr(a){ console.log(a*a); } sqr(6); function cube(a){ console.log(a*a*a); } cube(2); function len(s){ console.log(s.length); } len("Code"); function check(a){ console.log(a>0); } check(5); function greet(n){ console.log("Welcome "+n); } greet("Shubham");

Conclusion

  • Function saves time
  • Improves reusability
  • Makes code clean
